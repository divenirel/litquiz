# LitQuiz

LitQuiz — это веб-приложение на Django для изучения литературы в формате квиза.

Пользователь может:
- просматривать список вопросов;
- открывать страницу с деталями вопроса;
- добавлять новые вопросы;
- редактировать существующие вопросы;
- удалять вопросы;
- проходить квиз и получать результат с автоматической проверкой.

## Функциональность проекта

В приложении реализованы:
- главная страница;
- страница со списком вопросов;
- страница с деталями вопроса;
- форма добавления вопроса;
- форма редактирования вопроса;
- удаление вопроса с подтверждением;
- страница прохождения квиза;
- страница результата квиза;
- валидация формы;
- стилизация интерфейса с помощью Bootstrap.

## Технологии

- Python
- Django
- SQLite
- HTML
- Bootstrap 5

## Установка и запуск

git clone https://github.com/divenirel/litquiz.git

cd litquiz

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
